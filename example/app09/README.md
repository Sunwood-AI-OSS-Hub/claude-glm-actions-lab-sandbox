This application is a color picker tool that allows users to select colors using a native color picker or preset palette. It automatically converts colors between HEX, RGB, and HSL formats, providing real-time preview.

## Features âœ¨

- **Color Picker**: Select colors using the native color picker
- **HEX Input**: Direct HEX code input (supports 3-digit and 6-digit formats)
- **Color Conversion**: Automatic conversion from HEX to RGB and HSL
- **Preview**: Real-time preview of the selected color
- **Palette**: Quick color selection from frequently used colors
